1. Цель разработки
Разработать веб-сайт для поиска и сохранения мест для досуга (кино, театры, парки) в конкретном городе.

2. Технологический стек
Backend: Python + Flask

Frontend: HTML, CSS

Хранение данных: встроенные структуры Python (словари, списки) + сессии Flask

3. Функциональные требования
3.1. Регистрация и авторизация
Форма регистрации: логин, пароль, подтверждение пароля

Форма входа: логин, пароль

Выход из аккаунта

Сессия пользователя сохраняется до закрытия браузера

3.2. Меню сайта
Главная (список всех мест)

Куда сходить (с подменю или вкладки: Кино, Театры, Парки)

Личный кабинет (доступен только авторизованным)

Вход / Регистрация (скрывается после авторизации)

3.3. Категории мест (вкладки)
Кино — название кинотеатра, адрес, сеансы

Театры — название театра, адрес, афиша

Парки — название парка, адрес, часы работы, наличие активностей

3.4. Карточка места
Каждая карточка содержит:

Название

Категория

Адрес

Время работы

Кнопка «В избранное» (только для авторизованных)

3.5. Избранное
Только авторизованные пользователи могут добавлять/удалять места в избранное

Неавторизованным при нажатии показывается сообщение: «Войдите, чтобы добавить в избранное»

Список избранных мест отображается в личном кабинете

4. Требования к интерфейсу
4.1. Общие
Единый шаблон с хедером, футером и основным блоком

Адаптивная верстка (ПК, планшет, телефон)

Карточки мест расположены сеткой (3–4 колонки на ПК, 1 колонка на телефоне)

4.2. Цветовая схема
Основной фон: светлый или темный на выбор

Акцентный цвет: для кнопок и активных вкладок
